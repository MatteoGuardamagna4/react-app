## Setup
#### Option 1 (full features)
1. Clone the repo
2. Create a `.env` file in the root:
```
GROQ_API_KEY=your_key_here
GEMINI_API_KEY=your_key_here
```
3. **Run the app**
```
   npm install && npm run dev
```
